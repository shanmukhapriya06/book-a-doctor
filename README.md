🩺 Book a Doctor

A MERN Stack Healthcare Appointment Booking Platform that connects patients with verified doctors and enables seamless appointment management through a secure and user-friendly web application.

🚀 Features

 👤 Patient/User

* 🔐 Secure Registration & Login
* 👨‍⚕️ Browse Verified Doctors
* 📅 Book Appointments Online
* 📄 Upload Previous Prescriptions & Medical Documents
* 📋 View Appointment History
* 🔔 Receive Notifications

  🩺 Doctor

* 📝 Apply for Doctor Registration
* 📅 Manage Appointments
* ⏰ Update Availability
* 📂 View Patient Information & Uploaded Documents

  🛠️ Admin

* 👥 Manage Users
* ✅ Approve/Reject Doctor Applications
* 📊 Monitor Appointments
* 🏥 Manage Platform Operations

 ⚙️ Installation

 📥 Clone Repository


git clone https://github.com/shanmukhapriya06/book-a-doctor.git
cd book-a-doctor


🖥️ Backend Setup

cd server
npm install

Create a .env file:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=8001

Run Backend:

npm start

🌐 Frontend Setup

cd client
npm install
npm run dev
